# CeloHT Finance & Tokenomics

**CeloHT is not a cryptocurrency or token.**  
It is a decentralized platform leveraging **Celo cUSD** via **Valora**, focused on three pillars:

1. Education
2. Agents
3. Reforestation

This repository contains the **financial principles, funding, and utilization model** of CeloHT.

- `FUNDING.md` → Funding policy and allocation principles
